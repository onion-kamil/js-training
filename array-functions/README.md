## About

**Cel zadania:**

- Zaimplementuj własne wersje funkcji tablicowych Array.prototype.length oraz Array.prototype.filter, bez wykorzystania tych wbudowanych.

**Przykład:**

```javascript
filter([1, 2, 3, 4], isEven); // => [2, 4]
length([1, 2, 3, 4]); // => 4
```
